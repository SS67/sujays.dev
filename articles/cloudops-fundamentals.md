# CloudOps Fundamentals

*Managing cloud infrastructure at scale*

## Overview

CloudOps is the practice of managing and operating cloud infrastructure with a focus on automation, reliability, and efficiency.

## Core Principles

### Infrastructure as Code
Everything is version-controlled — no click-ops.

### Immutable Infrastructure
Replace, don't patch. Build new, destroy old.

### GitOps
Git as the single source of truth for infrastructure state.

### Observability
Metrics, logs, and traces — know what's happening.

# Identity Management with Red Hat IdM

*Centralized authentication and authorization*

## Overview

Red Hat Identity Management (IdM) provides centralized authentication, authorization, and account information by storing data about users, groups, hosts, and other objects.

## Key Features

- Kerberos authentication
- LDAP directory
- DNS management
- Certificate authority
- Host-based access control
- Sudo rule management
- Active Directory trust

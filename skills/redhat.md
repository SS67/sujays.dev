# Red Hat Ecosystem

## RHEL (Red Hat Enterprise Linux)

Enterprise Linux distribution — stability, security, long-term support.

### Core Skills
- System installation and kickstart
- Package management (dnf/yum)
- Security hardening and compliance
- Performance tuning
- Troubleshooting and recovery

---

## Red Hat Satellite

Lifecycle management for RHEL systems at scale.

### Capabilities
- Content management (repos, packages, errata)
- Provisioning and kickstart
- Configuration management
- Patch management
- Compliance reporting

### Architecture
- Satellite Server (central)
- Capsule Servers (distributed)
- Content Views and Lifecycle Environments

→ See: [/knowledge/redhat/satellite.md](../knowledge/redhat/satellite.md)

---

## Red Hat IdM (Identity Management)

Centralized identity, authentication, and policy.

### Features
- User and group management
- Kerberos authentication
- DNS and certificate management
- Host-based access control
- Sudo rule management
- Trust with Active Directory

→ See: [/knowledge/redhat/idm.md](../knowledge/redhat/idm.md)

---

## Ansible

Agentless automation for configuration management.

### Experience
- Playbook development
- Role-based structure
- Inventory management (static/dynamic)
- Ansible Vault for secrets
- AWX/Tower for enterprise use

→ See: [/knowledge/ansible/](../knowledge/ansible/)

## OpenShift

Red Hat's enterprise Kubernetes platform.

### Key Differentiators
- Integrated CI/CD (Pipelines, GitOps)
- Developer console
- OperatorHub
- Built-in monitoring and logging
- Multi-cluster management (ACM)
